def pushDominoes(self, dominoes):
    n为字符串dominoes的长度
    初始化队列q
    定义数组time，长度等于字符串dominoes的长度，数组元素初值均为-1，time[i]表示第i张骨牌翻倒的时间
    定义长度为n的字符串数组forece，force[i]表示第i张骨牌的受力字符串
    i为字符串下标，初值为0；f为字符串dominoes中下标为i的字符
    while(对字符串进行遍历)
        if (当前遍历字符不为'.')
            将当前遍历下标加入队列
            第i张骨牌翻倒的时间等于0
            将当前遍历字符添加至第i张骨牌的受力字符串
        else:
        字符串下标加一，更新当前遍历下标对应的当前遍历字符
    定义长度为n数组res，数组元素初值均为'.'
    while(队列q不为空)
        定义i等于队头元素，将队头元素出队
        if (!(第i张骨牌的受力字符串的长度等于1))
            continue
        令数组res中下标为i的元素等于第i张骨牌的受力字符串中第一个字符
        定义字符f等于第i张骨牌的受力字符串中第一个字符
        定义变量ni，如果字符f为'L'，ni等于i减一，否则ni等于i加一
        if (!(ni大于等于0且字符串dominoes的长度大于0))
            continue
        定义变量t等于第i张骨牌翻倒的时间
        if (第ni张骨牌翻倒的时间等于-1)
            将ni加入队列
            第ni张骨牌翻倒的时间等于t加一
            将字符f添加至第ni张骨牌的受力字符串
            continue
        else:
            if (!(第ni张骨牌翻倒的时间等于t加一))
                continue
            将字符f添加至第ni张骨牌的受力字符串
            continue
    output:  将数组res转化为字符串并返回
