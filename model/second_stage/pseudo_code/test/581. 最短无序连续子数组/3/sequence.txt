def findUnsortedSubarray(self, nums):
    n为数组nums的长度。
    初始化当前最大元素maxn等于无穷小，初始化右界right等于-1
    初始化当前最小元素maxn等于无穷大，初始化左界right等于-1
    i等于0
    while(i小于数组长度)
        if (当前最大元素大于数组中下标为i的元素)
            右边界等于i
        else:
            当前最大元素等于数组中下标为i的元素
        if (当前最小元素小于数组中下标为n减i减1的元素)
            左边界等于n减i减1
        else:
            当前最小元素等于数组中下标为n减i减1的元素
        i自增1
    output:  如果右界等于-1则返回0，否则，返回左右界之差加1
