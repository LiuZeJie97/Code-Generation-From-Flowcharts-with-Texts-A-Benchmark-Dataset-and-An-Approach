def findUnsortedSubarray(self, nums):
    n为数组nums的长度。
    初始化右界right等于0，
    使用max_num，保存到当前位置为止的最大值，初值为数组第一个元素
    i等于0
    while(i小于数组长度)
        if (若数组下标为i的元素大于等于当前位置为止的最大值)
            更新当前位置为止的最大值等于数组下标为i的元素
        else:
            更新右界等于i
        i自增1
    初始化左界left数组长度
    初始化min_num，保存到当前位置为止的最小值，初值为数组最后一个元素
    i等于数组长度减一
    while(i小于数组长度)
        if (若数组下标为i的元素小于等于当前位置为止的最小值)
            更新当前位置为止的最小值等于数组下标为i的元素
        else:
            更新左界等于i
        i自减1
    output:  如果左右界之差加1大于0，返回左右界之差加1，否则，返回0
