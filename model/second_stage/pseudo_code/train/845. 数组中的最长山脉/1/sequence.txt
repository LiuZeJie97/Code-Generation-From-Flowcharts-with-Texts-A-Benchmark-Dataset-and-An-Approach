def longestMountain(self, a):
    if (数组a为空)
        output:  返回0
    else:
        n为数组a的长度
        定义数组left，其长度等于数组a的长度，初值均为0
        i为数组下标，初值为1
        while(当i小于n)
            如果数组a中下标为i-1的元素小于下标为i的元素，则数组left中下标为i的元素等于下标为i-1的元素加一，否则该元素等于0
            i自增1
        定义长度为n的数组right，其初值均为0
        i为数组下标，初值为数组a的长度减2
        while(当i大于-1)
            如果数组a中下标为i+1的元素小于下标为i的元素，则数组right中下标为i的元素等于下标为i+1的元素加一，否则该元素等于0
            i自减1
        定义变量ans等于0，用于记录答案
        i为数组下标
        while(当i小于n)
            if (数组left中下标为i的元素大于0且果数组right中下标为i的元素大于0)
                如果它们的和加一大于答案，则将答案更新为它们的和加一
            else:
            i自增1
        output:  返回答案
