def convert(self, s, numRows):
    定义n和r，其中，n等于字符串s的长度，r等于给定的行数numRows
    if (给定一行或给定的行数大于字符串的长度)
        output:  直接返回字符串本身
    else:
        周期t等于行数乘2减2
        矩阵的列数c等于(字符串的长度加周期减去1)整除(周期乘(行数减1))
        定义二维矩阵mat，初值为''，第一维长度为行数r，第二维长度为列数
        设当前填写的位置为(x,y)，即二维矩阵mat的x行y列。初始(x,y)等于(0,0)
        i为当前字符下标，初值等于0，ch为字符串中的一个字符
        while(遍历字符串每一个字符)
            在当前位置上填写该字符
            if (若当前字符下标满足i对t取模 小于r减去1)
                当前填写的位置向下移动
            else:
                当前填写的位置向右上移动
            字符下标右移，同时更新ch
        output:  填写完成后，逐行扫描矩阵中的非空字符，组成答案
