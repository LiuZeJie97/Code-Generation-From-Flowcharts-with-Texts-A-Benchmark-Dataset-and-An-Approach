def convert(self, s, numRows):
    定义n和r，其中，n等于字符串s的长度，r等于给定的行数numRows
    if (给定一行或给定的行数大于字符串s的长度)
        output:  直接返回字符串本身
    else:
        定义Z字形变换的周期t等于行数乘2再减2
        使用列表ans保存新的字符串
        定义矩阵的行的下标为i，初值等于0
        while(行的下标小于给定的行数)
            j等于0，为每个周期的起始下标
            while(周期的起始下标小于字符串的长度减去给定的函数)
                当前周期的第一个字符的下标等于起始下标加上当前行的下标，将第一个字符拼接在列表末尾
                if (0小于当前行小于给定的行数，且起始下标加上周期减去当前行的下标小于字符串长度)
                    当前周期的第二个字符的下标等于起始下标加上周期减去当前行的下标，将第二个字符拼接在列表末尾
                else:
                周期的起始下标j等于其自身减去Z字形变换的周期
            矩阵的行的下标加1
        output:  返回新的字符串
