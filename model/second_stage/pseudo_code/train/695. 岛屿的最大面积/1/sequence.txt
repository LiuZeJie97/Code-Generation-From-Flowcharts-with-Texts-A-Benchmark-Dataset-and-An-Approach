def maxAreaOfIsland(self, grid):
    定义变量ans等于0，用于记录最大岛屿面积
    i为横坐标，l为二维数组grid中当前横坐标对应的数组
    while(对该二维数组进行遍历)
        j为纵坐标，n为数组l中当前纵坐标对应的元素
        while(对该数组进行遍历)
            定义变量cur记录现在的岛屿面积，初值为0
            定义队列q，将位置坐标(i,j)入队
            while(当队列不为空)
                定义cur_i等于队头元素的横坐标，表示现在的横坐标，定义cur_j等于队头元素的纵坐标，表示现在的纵坐标，并将队头元素出队
                if (如果现在的横坐标小于0或现在的纵坐标小于0或现在的横坐标等于数组grid的行数或现在的纵坐标等于数组grid的列数)
                    continue
                if (如果数组grid中cur_i行cur_j列处的元素不为1)
                    continue
                现在的岛屿面积加一
                数组grid中cur_i行cur_j列处的元素等于0
                di为横坐标方向向量，dj为纵坐标方向向量，(di,dj)表示方向向量
                while((di,dj)的取值分别为(0,1),(0,-1),(1,0),(-1,0))
                    定义下一步横坐标next_i等于现在的横坐标加当前横坐标方向向量，下一步纵坐标next_j等于现在的纵坐标加当前纵坐标方向向量
                    将现在的横坐标、纵坐标组成的位置坐标入队
            最大岛屿面积等于它与现在的岛屿面积之间的最大值
            更新纵坐标及其对应的元素
        更新横坐标及其对应的数组
    output:  返回最大岛屿面积
