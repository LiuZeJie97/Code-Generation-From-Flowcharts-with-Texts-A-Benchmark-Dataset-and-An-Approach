def findLongestChain(self, pairs):
    对数对集合pairs排序
    定义数组dis，其长度等于数对集合pairs的长度，初值均为1，其中dis[i]用于存储以下标为i的数对结尾的最长链的长度
    n为数组下标
    while(该数组下标小于数对集合的长度)
        m为数组下标
        while(该数组下标小于数组下标m)
            if (下标为m的数对中的最后一个元素小于下标为n的数对中的第一个元素)
                将以下标为n的数对结尾的最长链的长度更新为它与以下标为m的数对结尾的最长链的长度值加一之间的最大值
            else:
            该数组下标加一
        该数组下标加一
    output:  返回数组dis中的最大值
