def isPerfectSquare(self, num):
    定义左边界start，初值为0；定义右边界end，初值为给定正整数num
    while(左边界小于等于右边界)
        定义中值center等于左边界与右边界的和整除2
        定义变量square等于中值的平方
        if (square小于给定正整数)
            左边界等于中值加一
            continue
        else:
            if (!(square大于给定正整数))
                output:  给定正整数是完全平方数
            右边界等于中值减一
            continue
    output:  给定正整数不是完全平方数
