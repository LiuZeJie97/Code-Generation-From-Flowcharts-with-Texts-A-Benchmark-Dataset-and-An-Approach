def new21Game(self, n, k, maxPts):
    if (如果k等于0)
        output:  爱丽丝的分数不超过n的概率是1.0
    else:
        定义长度等于k+maxPts，数组元素初值均为0的数组dp，dp[i]表示从得分为i开始游戏且分数不超过n的概率
        i为数组下标，初值为k
        while(计算n与k+maxPts-1之间的最小值，当i小于该计算结果加一时)
            从得分为i开始游戏且分数不超过n的概率等于1.0
            i等于i加一
        计算n-k+1与maxPts之间的最小值，令float型的该最小值除以maxPts，从得分为k-1开始游戏且分数不超过n的概率等于该计算结果
        i为数组下标，初值为k-2
        while(当i大于-1)
            计算从得分为i+maxPts+1开始游戏且分数不超过n的概率与从得分为i+1开始游戏且分数不超过n的概率的差值，令该差值除以maxPts，从得分为i开始游戏且分数不超过n的概率等于从得分为i+1开始游戏且分数不超过n的概率与该计算结果的差值
            i等于i减一
        output:  返回从得分为0开始游戏且分数不超过n的概率
