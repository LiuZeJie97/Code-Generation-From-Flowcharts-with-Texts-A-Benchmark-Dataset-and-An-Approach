def checkIfExist(self, a):
    对数组a排序
    右指针q等于0
    左指针p等于0
    while(左指针小于数组a的长度)
        while(右指针q小于数组的长度且数组左指针元素的2倍大于右指针元素)
            将右指针右移1
        if (右指针不等于数组的长度且左指针不等于右指针且数组左指针元素的2倍等于右指针元素)
            output:  整数及其两倍数存在
        将左指针右移1
    右指针等于数组a的长度减一
    左指针p等于数组的长度减一
    while(左指针大于-1)
        while(右指针q大于-1且数组左指针元素的2倍小于右指针元素)
            将右指针左移1
        if (右指针不等于-1且左指针不等于右指针且数组左指针元素的2倍等于右指针元素)
            output:  整数及其两倍数存在
        将左指针左移1
    output:  整数及其两倍数不存在
