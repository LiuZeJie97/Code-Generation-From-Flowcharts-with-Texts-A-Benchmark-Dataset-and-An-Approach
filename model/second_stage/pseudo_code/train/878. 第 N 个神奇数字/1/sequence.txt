def nthMagicalNumber(self, n, a, b):
    定义模数MOD等于10的9次方+7
    定义变量l等于a整除a与b的最大公约数再乘b
    定义下边界等于0
    定义上边界等于10的15次方
    while(当下边界小于上边界)
        定义中值mid等于上边界与下边界的和整除2
        定义变量m等于中值整除a加上中值整除b减去中值整除l
        if (如果m小于n)
            将下边界更新为中值加一
            continue
        else:
            将上边界更新为中值
            continue
    output:  返回下边界对模数取模的结果
