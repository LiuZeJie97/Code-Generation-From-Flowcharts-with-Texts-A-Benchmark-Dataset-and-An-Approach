def fallingSquares(self, positions):
    初始化数组qans，其长度等于数组positions的长度，初值均为0
    i为数组下标，当前遍历数对(left, size)为数组positions中当前数组下标处的元素
    while(对数组进行遍历)
        定义变量right等于当前遍历数对中两个数的和
        数组qans中下标为i的元素等于size
        j为数组下标，其初值为i+1
        while(该数组下标小于数组positions的长度)
            定义left2等于当前遍历数组下标为j的元素的第一项，size2等于当前遍历数组下标为j的元素的最后一项，
            定义变量right2等于left2与size2的和
            if (如果left2小于right且left小于right2)
                数组qans中下标为j的元素等于它与下标为i的元素之间的最大值
            else:
            该数组下标加一
        当前数组下标加一，并将当前遍历元素更新当前数组下标对应位置处的元素
    定义数组ans，表示堆叠高度列表
    遍历元素x用于遍历数组qans中的元素，初值为被遍历数组的第一项
    while(对数组遍历)
        if (如果堆叠高度列表不为空)
            计算堆叠高度列表中最后一项与当前元素之间的最大值，将该值添加进堆叠高度列表
        else:
            将当前元素添加进堆叠高度列表
        将当前元素更新为数组中的下一项
    output:  返回堆叠高度列表
