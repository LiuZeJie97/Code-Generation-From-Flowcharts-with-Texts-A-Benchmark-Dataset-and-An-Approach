def numDecodings(self, s):
    n为字符串s的长度
    初始化向量f，长度为字符串长度加1，f[0]等于1，其余各位等于0。设f[i]表示字符串的前i个字符s[1..i]的解码方法数
    i等于1，为字符串下标
    while(字符串下标小于向量f的长度)
        if (字符串下标指向字符的前一个字符不等于"0")
            前i个字符的解码方法数等于其自身加上前i-1个字符的解码方法数
        else:
        if (当前下标大于1，且字符串s[i-2]不等于"0"，且s[i-2:i]小于或等于数字26)
            前i个字符的解码方法数等于其自身加上前i-2个字符的解码方法数
        else:
        下标右移一个单位
    output:  返回字符串的解码方法总数
