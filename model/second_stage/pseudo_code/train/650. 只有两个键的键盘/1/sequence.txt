def minSteps(self, n):
    初始化长度为n+1的一维数组f，其初值均为0，f[i]表示打印出i个A的最少操作次数
    i为数组下标，其初值为2
    while(数组下标小于数组f的长度加一)
        将打印出i个A的最少操作次数初始化为float型的inf
        定义变量j，初值为1
        while(j的平方小于i时)
            if (i可以整除j)
                计算打印出j个A的最少操作次数加上i整除j的结果，令打印出i个A的最少操作次数等于它自身与上述计算结果之间的最小值
                计算打印出i整除j个A的最少操作次数加上j的结果，令打印出i个A的最少操作次数等于它自身与上述计算结果之间的最小值
            else:
            j加上一
        将当前数组下标的值更新为当前数组下标加一
    output:  返回打印出n个A的最少操作次数
