def wordBreak(self, s, wordDict):
    字符串s的长度为n。
    初始化dp=[False, ,False]，长度为字符串的长度加1。dp[i]表示字符串的前i位是否可以用wordDict中的单词表示。
    空字符可以用wordDict中的单词表示。
    i等于0，为遍历开始索引
    while(开始索引小于字符串s的长度)
        j为遍历结束索引，初值等于开始索引加1
        while(结束索引小于字符串长度加1)
            if (若s的前i位可以用wordDict中的单词表示 且从开始索引到结束索引的子串在wordlist中：)
                s的前j位可以用wordDict中的单词表示
            else:
            结束索引右移1个单位
        开始索引右移1个单位
    output:  返回dp最后一个元素
