def minRefuelStops(self, target, startFuel, stations):
    初始化数组dp，长度为数组stations的长度加一，数组元素初值均为0，dp[i]表示加i次油能走的最远距离
    加0次油能走的最远距离等于最初的燃料数startFuel
    定义初值为0的变量i为数组下标，station为数组stations中下标为i的子数组
    while(对该数组进行遍历)
        定义变量location等于当前遍历子数组中第一项元素，表示当前加油站的位置；变量capacity当前遍历子数组中最后一项元素，表示当前加油站的燃料数
        t为数组下标，初值为i
        while(当t大于-1)
            if (加t次油能走的最远距离大于当前加油站的位置)
                计算加t次油能走的最远距离与当前加油站的燃料数的和，加t+1次油能走的最远距离等于它与上述计算结果之间的最大值
            else:
            t等于t减一
        当前数组下标加一，当前遍历子数组移动至数组中下一个子数组
    定义变量ans，初值为-1，用于记录最低加油次数
    定义初值为0的变量i为数组下标
    while(i小于数组dp的长度)
        if (加i次油能走的最远距离大于目的地target)
            将最低加油次数的值更新为i
            break
        i等于i加一
    output:  返回最低加油次数
