def hIndex(self, citations):
    n为数组citations的长度，tot用于记录总引用数，初值为0
    定义数量数组counter用于记录当前引用次数论文的数量，其长度为数组citations的长度加一，数组元素初值均为0
    定义c为数组citations中的元素，表示当前论文被引用的次数
    while(对数组进行遍历)
        if (当前论文被引用的次数大于遍历数组的长度)
            数量数组中下标为n的元素加一
        else:
            数量数组中下标为c的元素加一
        更新当前论文被引用的次数
    h等于0，用于保存h指数
    定义i为数组下标，初值为数组citations的长度
    while(当数组下标大于-1)
        总引用数加上数量数组中下标为i的元素
        if (总引用数大于当前数组下标)
            h指数等于当前数组下标
            break
        数组下标减一
    output:  返回h指数
