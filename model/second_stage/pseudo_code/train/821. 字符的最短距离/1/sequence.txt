def shortestToChar(self, s, c):
    n为字符串s的长度
    定义长度等于字符串的长度，数组元素初值均为0的答案数组ans
    定义变量idx，其初值等于负的字符串的长度
    i等于0，表示为字符串索引；ch为字符串s中当前索引处的字符
    while(对字符串进行遍历)
        if (若当前索引处的字符等于字符c)
            idx等于当前字符串索引
        else:
        答案数组中当前字符串索引对应的元素等于当前字符串索引减去idx
        字符串索引加一，当前索引处的字符等于字符串中的下一个字符
    将idx更新为二倍字符串的长度
    i为字符串索引，初值为字符串s的长度减一
    while(字符串索引大于-1)
        if (若字符串中当前字符串索引对应的字符等于字符c)
            idx等于i
        else:
        答案数组中当前字符串索引对应的元素等于它与idx减去当前字符串索引之间的最小值
        字符串索引减1
    output:  返回答案数组
