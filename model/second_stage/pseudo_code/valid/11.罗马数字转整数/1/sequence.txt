def romanToInt(self, s):
    初始化字典VALUE_SYMBOLS={'I': 1,'V': 5,'X': 10,'L': 50,'C': 100,'D': 500,'M': 1000}，该向量由若干元组组成，每个元组是个(数值，字符)对，向量中的元组按数值大小由低到高排序，
    n等于罗马数字s的长度
    使用ans用于记录罗马数字对应的值，初值为0
    i为罗马数字下标，ch为罗马数字中的每一个字符
    while(对于每一个下标和每一个字符)
        使用value保存当前字符对应的值
        if (若当前字符小于罗马数字的长度减去1且当前字符对应的值小于当前字符右侧的字符对应的值)
            罗马数字对应的值需要减去当前字符对应的值
        else:
            罗马数字对应的值需要加上当前字符对应的值
        更新罗马数字的下标和字符
    output:  返回罗马数字对应的值
