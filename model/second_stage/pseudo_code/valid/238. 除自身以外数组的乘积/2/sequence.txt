def productExceptSelf(self, nums):
    length为数组nums的长度
    定义L, R, answer。其中，L 和 R 分别表示左右两侧的乘积列表，answer为全零向量
    定义L[i] 为索引 i 左侧所有元素的乘积，对于索引为 '0' 的元素，因为左侧没有元素，所以 L[0]等于1
    索引i等于1
    while(索引小于数组长度)
        索引 i 左侧所有元素的乘积等于数组num中下标为i减去1的元素乘上其左侧的所有元素的乘积
        索引向右移动一个单位
    R[i] 为索引 i 右侧所有元素的乘积，对于索引为 'length-1' 的元素，因为右侧没有元素，所以 R[length-1]等于1
    索引等于1
    while(索引小于数组长度减1)
        索引右侧所有元素的乘积等于num中下标为i加上1的元素乘上其右侧的所有元素的乘积
        索引向右移动一个单位
    索引i等于1
    while(索引小于数组长度减1)
        对于索引 i，除 nums[i] 之外其余各元素的乘积就是左侧所有元素的乘积乘以右侧所有元素的乘积
        索引向右移动一个单位
    output:  返回answer
