def longestPalindromeSubseq(self, sequence):
    length为字符串sequence的长度
    定义n行n列初值均为0的二维数组dp，其中dp[i][j]表示字符串sequence下标范围[i,j]内的最长回文子序列的长度
    m为数组下标，初值为字符串sequence的长度减一
    while(当前数组下标大于-1)
        令字符串下标范围[m,m]内的最长回文子序列的长度为1
        n为数组下标，初值为m+1
        while(当前数组下标小于字符串sequence的长度)
            if (字符串中下标为m的字符和下标为n的字符相同)
                令字符串下标范围[m,n]内的最长回文子序列的长度等于下标范围[m+1,n-1]内的最长回文子序列的长度加2
            else:
                求下标范围[m+1,n]内的最长回文子序列的长度与下标范围[m,n-1]内的最长回文子序列的长度之间的最大值，令字符串下标范围[m,n]内的最长回文子序列的长度等于该最大值
            将当前数组下标的值更新为上一个数组下标加一
        将当前数组下标的值更新为上一个数组下标减一
    output:  返回字符串下标范围[0,n-1]内的最长回文子序列的长度
