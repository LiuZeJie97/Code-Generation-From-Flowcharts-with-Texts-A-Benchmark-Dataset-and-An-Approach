def trap(self, height):
    定义用于记录答案的变量ans，初值为0
    left为左指针，初值为0；right为右指针，初值为数组height的长度减一
    定义变量leftMax用于记录左指针移动时出现的最大值，变量rightMax用于记录右指针移动时出现的最大值，其初值均为0
    while(左指针left小于右指针right)
        若数组height左指针处的值大于左指针移动时出现的最大值，则将数组左指针处的值更新为该最大值
        若数组右指针处的值大于右指针移动时出现的最大值，则将数组右指针处的值更新为该最大值
        if (数组左指针left处的值小于右指针right处的值)
            令答案ans加上左指针移动时出现的最大值和数组左指针处的值之间的差值
            左指针加一
            continue
        else:
            令答案ans加上右指针移动时出现的最大值和数组右指针处的值之间的差值
            右指针减一
            continue
    output:  返回答案
