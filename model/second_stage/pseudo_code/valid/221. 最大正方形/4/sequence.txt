        i等于0
        while(i小于矩形行数)
            j=0
            while(j小于列数)
                if (矩形下标为i，j的元素为1)
                    如果最大边长小于1，则更新为1
                    定义可能的最大正方形边长currentMaxSide等于行数减去i和列数减去j中的较小值
                    k等于1
                    while(k小于可能的最大正方形边长)
                        定义标记flag，初值为True
                        if (矩形第i加k行第j加k列的元素等于 '0')
                            break
                        m等于0
                        while(m小于k)
                            if (矩形第i加k行第j加m列的元素等于 '0' 或者 矩形第i加m行第j加k列的元素等于 '0')
                                更新标记为False
                                break
                            m自增1
                        if (!(标记为True))
                            break
                        如果最大边长小于k加1，则更新最大边长为k加1
                        k自增1
                else:
                j自增1
            i自增1
        最大正方形面积等于最大边长的平方
        output:  返回最大正方形面积
